## 🔧 Customization Guide

### Change Colors
Edit `tailwind.config.js`:
```js
colors: {
  brand: {
    blue: '#YOUR_COLOR',
    yellow: '#YOUR_COLOR',
  }
}
```

### Add Pages
1. Create `src/pages/NewPage.jsx`
2. Add route in `src/App.jsx`
3. Add link in `Navbar.jsx`

### Update Content
- Edit page components directly
- All content is in JSX
- Easy to find and modify

### Add Images
1. Place in `public/` folder
2. Reference as `/image.png`
3. Use responsive sizes

## 🐛 Troubleshooting

### Build Errors
```bash
rm -rf node_modules dist
npm install
npm run build
```

### Port in Use
```bash
# Change port
npm run dev -- --port 3001
```

### Images Not Loading
- Check paths start with `/`
- Clear browser cache
- Verify files in `public/`

### Styling Issues
- Ensure Tailwind classes are correct
- Check browser DevTools
- Verify tailwind.config.js
